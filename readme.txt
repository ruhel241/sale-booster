Supercharge your WordPress WooCommerce site with showing countdown timer for discount.

== Description ==

Sale Booster for WooCommerce is the ultimate WooCommerce plugin that allows you to create urgency and scarcity by providing countdown timer of your campaign or discount offer.
Urgency and scarcity are both powerful psychological triggers that will make more sales on your WooCommrece store and increase your conversion rate.

### Clear Deadline To Customer (Calendar Deadline )
Wp sales booster allows clear deadline to your shoppers . So  a website owner can  create a campaign targeting a specific deadline .Any one can set a campaign with coupon or special note with this plugin for a special occasion. Such as Black friday ,Critsmash ,cyber monday .

### Attention Grabbing Sticky Header Or Footer
This features allow to grab the attention of your user by releasing special note with the countdown timer . As example special note can be Next Day Delivery , Free Shipping , Discount Code etc .

### Show Specific Note Below Cart
Woocommarce website owner can release special note for individual products below cart option here you can place your discount coupon for any specific promotion offer  with timer .

### Remove Add To Cart Button
Sometimes Woocommarce website owner needs to remove the add to cart button for any specific product . This plugin allow this option to user .

### Hide Price
Woocommarce website owner  can hide price for any specific product . This plugin allow users to hide price for any specific product
